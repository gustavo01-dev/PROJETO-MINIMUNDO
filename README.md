 
# FightLab – Scripts SQL

Este repositório contém os scripts SQL utilizados na Experiência Prática 4 do projeto FightLab, incluindo comandos de inserção, consulta, atualização e remoção de dados.

## Estrutura dos Arquivos

- **fightlab_inserts.sql**  
  Script responsável pelo povoamento das tabelas principais do banco de dados.

- **fightlab_selects.sql**  
  Consultas SQL utilizando JOIN, WHERE, ORDER BY e LIMIT.

- **fightlab_updates.sql**  
  Comandos UPDATE aplicados de forma segura e condicional.

- **fightlab_deletes.sql**  
  Remoções condicionais garantindo a integridade dos dados.

## Como Executar

1. Abra o PGAdmin, Workbench ou outro cliente SQL.
2. Certifique-se de que o banco de dados e as tabelas foram criados previamente.
3. Execute os scripts na seguinte ordem:
   - `fightlab_inserts.sql`
   - `fightlab_selects.sql`
   - `fightlab_updates.sql`
   - `fightlab_deletes.sql`

## Requisitos

- Compatível com PostgreSQL.
- O modelo lógico deve estar devidamente criado antes da execução.
